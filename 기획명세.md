### 닉네임을 입력할때 유효성 검사

- 2~10자 이내 (입력하는 동안 실시간으로 글자 수 표시 (3/10))
- 한글·영문·숫자만 사용 가능
- 중복 체크는 입력을 멈춘 후 0.5초 정도 딜레이 후 자동 실행 (debounce) 불가하면 완료 누를 시 중복 체크

### 실시간 피드백 표시 문구

- ✓ 초록색: "사용 가능한 닉네임이에요"
- ✗ 빨간색: "이미 사용 중인 닉네임이에요"
- ✗ 빨간색: "2-10자로 입력해주세요"
- ✗ 빨간색: "한글, 영문, 숫자만 사용 가능해요"

### 버튼 상태

- 유효하지 않을 때: [시작하기] 버튼 비활성화 (회색)
- 유효할 때: [시작하기] 버튼 활성화 (포인트 컬러)
